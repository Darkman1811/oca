Introduction:
Ce repository git contient les tps réalisés dans le cadre de la préparation de l'examen oca java 8. 
L'objectif est d'effectuer tous les exercices pratiques du livre "java 8 fundamentals".
Ainsi que les tps java standard et OCA des sites "www.java2s.com" et "www.tutorialspoint.com".

Methodologie:
   1-Premiére lecture et surlignage des mots clés en orange.
   2-Deuxiéme lecture et étéblissement d'un résumé des mots clés sur fiche cartonnée.
   3-Pratique de tous les tps sités dans le livre.
   5-Pratique de tous les tps sur les sites "www.java2s.com" et "www.tutorialspoint.com".
   6-Effectuer les Mock Exams et revue des livres.
   7-Passer l'exament.
